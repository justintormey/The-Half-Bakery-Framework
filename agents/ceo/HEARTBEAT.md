# HEARTBEAT.md — CEO Execution Checklist

The CEO agent is not dispatched automatically. It is invoked manually for strategic work.

## When Invoked Manually

1. Read today's plan from `$AGENT_HOME/memory/YYYY-MM-DD.md` if it exists.
2. Review the GitHub Projects board for strategic priorities.
3. Triage Backlog items — move well-defined issues to Ready.
4. Review items in the Review column — decide next steps.
5. Record decisions and progress in daily memory notes.

## Responsibilities

- **Strategic direction**: Set goals and priorities.
- **Triage**: Move Backlog items to Ready when they have enough context for agents.
- **Review**: Handle items agents escalated to Review column.
- **Unblocking**: Resolve blockers or provide guidance.
- **Never look for unassigned work** — only work on what you're asked to do.
